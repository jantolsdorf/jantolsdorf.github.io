A Collection of Tools

A collection of tools that I find useful for my research and teaching.

Literature Management

  • Convert PDF-Paper to HTML: Paper to HTML
  • Collect and organize your papers, share libraries, plugins available for Word, Google Docs, Overleaf: Zotero
  • Colloborative Systematic Literature Reviews, supports in early steps of SLR, free: RAYYAN
  • Tidy your BibTex, free: Online BibTex Tidy
  • Ask questions about research papers: Google NotebookLLM

Conducting Studies

  • Recruiting participants
    • Prolific
  • Survey software
    • Very reliable and versatile survey software, free self-hosting, free basic online usage, can do a lot of customization with HTML5 and JS: LimeSurvey
    • Build your custom survey software with JS: SurveyJS

Quantitative Analysis

  • Free and in the Cloud
    • Jupyter Notebook in the Cloud, ready to go, be careful with personal data, awesome to get started with quantitative analysis: Anaconda Cloud
  • Free and installation required
    • IDE for Python: Anaconda; Can be used for R; Can be used in the browser; Recommend using with VSCodium
    • IDE for R: RStudio
      • Why R?: R can be advantagous over Python because it has much better support for modern statistical analysis in social sciences and psychology (via packages). Especially advantagous when working with Likert-type data, conducting factor analysis, structural equation modelling and alike.
    • Conduct statistical power analysis: G*Power
  • Not free but useful
    • UI guided Structural Equation Modeling (PLS-SEM), free for students: SmartPLS

Qualitative Analysis

Transcribe Audio Recordings
  • Offline privacy friendly transcription: noScribe
Coding
  • Free
    • No installation, supports text and image data, collaborative working, server in the EU, step-by-step tutorials available: QCAmap
  • Paid
    • Supports advanced qualitative analysis, supports Grounded Theory approaches, overkill/pricey for small projects: MAXQDA
  • Not tested, but free
    • TAGUETTE
    • QualCoder
    • CATMA

Data Visualization

  • Best no-code online data visualization, no account, no installation, simply paste your Excel or CSV file and choose visualization: RawGraphs

Encrypted File Storage and File Exchange

Encrypted storing and exchanging of research data, including participant personal data, backups, etc.
  • Cloud folder encryption when working in teams (E.g., shared folder in Google Drive, Dropbox, NextCloud): Cryptomator
  • Local hard drive and volume encryption: VeraCrypt
  • Encrypt files to be stored or sent via your preferred channel: Hat.sh

Colloborative Working

  • Online whiteboard, awesome: Miro
  • Google Drive like privacy preserving office cloud, free/ donation based, no account required, awesome: CryptPad

More Tools

  • "Free and easy-to-use online PDF tools that make you more productive": PDF24

Make MacOS Usable

If only the quality of MacOS matched the production quality of Mac hardware ...
  • Usable MacOS Must Haves
    • Reverse scrolling when using non-Apple mice: Scroll Reverser
    • Fix Music app opens “randomly”/ on Bluetooth connection (when not having an Apple Account): noTunes
    • Windows like easy window management and resizing: Rectangle
  • Optional
    • Powr users of Alt-Tab: Alt-Tab MacOS
    • Windows VM: Parallels
