A Collection of Tools

A collection of tools that I find useful for my research and teaching.

Literature Management

Tool Description Cost
Paper to HTML Convert PDF papers to HTML for easier reading and accessibility. Free
Zotero Manage literature, share libraries, and use with Word, Google Docs, or Overleaf via plugins. Free
RAYYAN Supports collaborative systematic literature reviews, especially early-stage screening. Free
Online BibTex Tidy Clean and format your BibTeX files easily. Free
Google NotebookLM Ask questions and interact with your research papers using language models. Free

Survey Software & Participant Recruitment

Tool Description Cost
Prolific Participant recruitment platform. Paid
LimeSurvey Very reliable and versatile survey software, free self-hosting, free basic online usage, highly customizable with HTML5 and JS. Free / Paid
SurveyJS Build your own custom survey software with JavaScript. Free

Quantitative Analysis Tools

Tool Description Cost
Anaconda Cloud Jupyter Notebook in the cloud; ready to use for quantitative analysis. Be cautious with personal data. Free (Cloud-based)
Anaconda Python IDE (also usable for R); includes Jupyter; runs in browser. Recommended with VSCodium. Free (Install)
RStudio R IDE. R is strong for social science stats (e.g., Likert data, factor analysis, SEM). Free (Install)
G*Power Tool for statistical power analysis. Free (Install)
SmartPLS UI tool for PLS-SEM. Useful for structural equation modeling. Free for students. Paid (Free for students)
